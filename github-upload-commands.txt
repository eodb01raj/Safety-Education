# GitHub Upload Commands
# Run these commands in your project folder

# 1. Initialize Git repository
git init

# 2. Add all files
git add .

# 3. Make initial commit
git commit -m "Initial release - Alpha 0.01"

# 4. Add your GitHub repository as remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/fire-safety-training-app.git

# 5. Push to GitHub
git push -u origin main

# Note: You'll need to create the repository on GitHub first, then use these commands
